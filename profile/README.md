# 🤖 키즈프렌즈 시연 실행 가이드

> 이 문서대로 따라 하면 개발을 몰라도 시연을 켤 수 있습니다.
> **순서대로** 하세요. 각 단계 끝의 ✅ "이렇게 보이면 성공"을 꼭 확인하세요.

---

## 🗺️ 전체 그림

```
[라즈베리파이] --센서값--> [컴퓨터 A: 서버] <--질문/응답--> [테미 로봇]
                                  ▲
                          (테미 앱은 'B 컴퓨터'에서
                           테미에 설치만 해두면 끝)
```

| 장비 | 하는 일 | 시연 중 켜둬야 함? |
|---|---|---|
| **컴퓨터 A** | 서버(두뇌). 모두가 여기에 접속 | ✅ 계속 켜둠 |
| **라즈베리파이** | 센서/카메라로 아이 감지 | ✅ 계속 켜둠 |
| **테미 로봇** | 말하고 화면 보여주는 본체 | ✅ 계속 켜둠 |
| **컴퓨터 B** | 테미에 앱을 설치하는 용도 | ❌ 설치 후엔 꺼도 됨 |

---

## 🧰 시작 전 준비물

- **컴퓨터 A** 에 설치: Java(JDK 17), MySQL, ngrok, Git
- **컴퓨터 B** 에 설치: Android Studio, (테미 연결용) USB 케이블
- **라즈베리파이**: 파이썬(기본 설치돼 있음), 인터넷 연결
- 공통: 모든 장비가 **같은 와이파이** 또는 인터넷에 연결

> 📌 **용어 한 줄 사전**
> - **터미널/명령창**: 명령어를 입력하는 검은 창 (Windows=PowerShell, 라즈베리파이=터미널)
> - **ngrok**: 내 컴퓨터의 서버를 인터넷에서 접속 가능한 주소로 만들어 주는 도구
> - **주소(URL)**: `https://...ngrok...dev` 처럼 생긴 서버 접속 경로

---

## 1️⃣ 컴퓨터 A — 서버 켜기

1. MySQL이 켜져 있는지 확인합니다. (데이터 저장소)
2. 명령창을 열고 아래를 입력합니다.
   ```powershell
   cd KF_BE
   .\gradlew.bat bootRun
   ```
3. 잠시 기다리면 서버와 ngrok이 함께 켜집니다.
4. 화면에 나오는 **서버 주소(`https://...ngrok-free.dev`)를 메모**합니다. → 이걸 `[서버주소]`라고 부르겠습니다.

✅ **이렇게 보이면 성공**: 웹 브라우저에서 `[서버주소]/api/health` 를 열었을 때 정상 응답(에러 아님)이 보임.

---

## 2️⃣ 라즈베리파이 — 센서 켜기

1. 라즈베리파이 터미널을 엽니다.
2. 서버 주소를 알려준 뒤 실행합니다. (`[서버주소]`를 1단계에서 메모한 값으로 바꾸세요)
   ```bash
   cd KF_HW
   source .venv/bin/activate
   export KF_BE_URL="[서버주소]"
   export KF_ROBOT_ID="TEMI_01"
   python src/main.py
   ```

✅ **이렇게 보이면 성공**: 터미널에 `백엔드 전송 대상: [서버주소]/api/sensor-events` 가 출력됨.

> 🛟 카메라/센서 장비가 아직 없을 때(연습용): `python src/main.py` 대신
> ```bash
> python tests/mock_stream.py
> ```
> 를 실행하면 가짜 센서값으로 테스트할 수 있습니다.

---

## 3️⃣ 컴퓨터 B — 테미에 앱 설치 (한 번만)

1. **Android Studio**로 `KF-FE` 프로젝트를 엽니다.
2. 앱이 바라볼 **서버 주소를 1단계의 `[서버주소]`로 설정**합니다.
   (담당 개발자에게 "백엔드 주소 어디서 바꾸나요?"라고 물어보면 한 줄로 알려줍니다.)
3. 테미를 **USB 케이블로 컴퓨터 B에 연결**합니다.
4. Android Studio 상단의 **초록색 ▶(Run)** 버튼을 누릅니다.
5. 설치가 끝나면 테미 화면에 앱이 뜹니다.

✅ **이렇게 보이면 성공**: 테미 화면에 캐릭터 얼굴과 `"친구야" 하고 부르거나 화면을 터치해줘!` 안내가 보임.

> ⚠️ Android Studio에서 **Java/Gradle 버전을 바꾸지 마세요.** 그대로 두면 됩니다. (바꾸면 설치가 깨집니다.)
> 설치가 끝나면 컴퓨터 B는 꺼도 됩니다.

---

## 4️⃣ 시연 시작

테미 앞에서:
1. **"친구야"** 라고 부릅니다. → 테미가 대화를 시작합니다.
2. 질문/요청을 말합니다. 예) "퀴즈 풀고 싶어", "미끄럼틀 어디 있어?", "회원 등록하고 싶어"
3. 아이가 테미 앞에 다가가면(또는 센서 작동 시) 테미가 인사/반응합니다.

✅ **성공 신호**: 말하는 동안 테미 화면 **아래쪽에 실시간 자막**이 뜨고, 테미가 음성으로 답합니다.

## 🆘 문제가 생기면

| 증상 | 이렇게 해보세요 |
|---|---|
| 아무 것도 연결 안 됨 | 컴퓨터 A 서버가 켜져 있나? `[서버주소]/api/health` 다시 확인 |
| 주소가 자꾸 바뀜 | ngrok 주소가 바뀐 것. 새 주소를 2·3단계에 다시 넣기 |
| 테미가 서버를 못 찾음 | 테미가 와이파이에 연결됐는지, 앱 주소가 `[서버주소]`인지 확인 |
| 라즈베리파이가 서버로 안 쏨 | `export KF_BE_URL="[서버주소]"` 를 안 넣고 실행했을 수 있음. 다시 실행 |
| "친구야"를 못 알아들음 | 주변 소음 줄이기, 테미 언어가 **한국어**인지 확인 |
| 음성이 멈춤 | 테미 화면을 한 번 터치하거나 앱을 재시작 |
